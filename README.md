## ğŸ¨ Customization

### Changing Colors

Edit `css/style.css` - CSS Variables section:

```css
:root {
    --gold: #D4AF37;           /* Primary gold accent */
    --dark-gold: #B8860B;      /* Darker gold shade */
    --black: #0A0A0A;          /* Primary text */
    --cream: #FAF8F3;          /* Background */
    /* ... modify as needed */
}
```

### Adding/Removing Images

1. Add images to the `images/` folder
2. Update `index.html` in the Portfolio Gallery section:

```html
<div class="gallery-item">
    <img src="images/YOUR-IMAGE.jpeg" alt="Description">
    <div class="gallery-overlay">
        <h3>Title</h3>
        <p>Subtitle</p>
    </div>
</div>
```

### Updating Content

All content is in `index.html`. Simply search for the section you want to update and modify the text directly.
